## 1. 灵魂四问

对每个主题给出具体答案，禁止用“所有人”“传递价值”“引发共鸣”等空话。

1. **我对谁说？** 写出一个处境明确的人群，以及他们此刻的焦虑、欲望或误解。
2. **我想说什么？** 压成一句可证伪的主张。成片只服务这一句。
3. **希望对什么人产生什么影响？** 写明看完后希望发生的认知、情绪或行动变化。
4. **我的对立面是谁？** 优先写观念、惯性、误区、制度张力或旧自我；不要为了情绪凭空树敌或攻击弱者。

“对立面”用于制造叙事张力，不等于必须吵架。若原访谈没有敌意，不能剪出受访者并未表达的人身攻击。

## 2. 主题评分

每项 0–5 分，并附逐字稿证据。总分只用于比较，不代替编辑判断。

| 维度 | 5 分表现 |
|---|---|
| 单一击穿 | 一句话能说清，删去旁枝后仍成立 |
| 受众连接 | 明确命中一种处境、痛点或渴望 |
| 原文证据 | 有多处具体原句、故事或细节支撑 |
| 冲突张力 | 有清晰反差、误区、代价或立场碰撞 |
| 情绪能量 | 有语气、停顿、笑泪或真实反应可用 |
| 新鲜程度 | 不是正确废话，有反常识或独特经历 |
| 可回剪性 | 关键原句有连续语义和可靠时间码 |
| 伦理完整 | 不靠断章取义改变受访者本意 |

优先选择证据充分且可回剪的主题，而不是标题最炸但原文支撑最弱的主题。

## 3. “环环相扣，从一而终”终审

逐段回答：

- 本段边界在哪里？主题和目的是什么？
- 本段服务哪一条灵魂四问？不服务则删除或另成一条视频。
- 本段是否紧紧围绕唯一主张？
- 是否有抽象判断后的具体人、事、动作、场景或代价？
- 与前一段是什么关系：因果、转折、递进、解释、举例还是回扣？
- 调换前后能否让观众更快进入处境，再逐步接受结论？
- 有没有重复解释、结论抢跑、信息断层或无回报的铺垫？
- 最后一段是否完成主题，而不是突然停止？

推荐结构不是硬模板，但常见有效顺序是：钩子/冲突 → 处境 → 关键细节 → 认知转折 → 代价或证明 → 回扣主张 → 余味/互动。

## 4. 流量压力测试

用“为什么可能没人看”寻找真实弱点，不把算法当万能解释：

- 前 3–8 秒是否只有背景，没有问题、冲突、结果或异常？
- 标题承诺是否在正文得到原文证据？
- 是否只有观点，没有细节、画面和代价？
- 是否把最强信息埋得太后？
- 是否每段都在推进，还是换句式重复？
- 是否只有愤怒，没有祝福、理解或有效落点？
- 是否因断章取义造成“炸”但不可信？
- 结尾是否收束或留下具体问题，而不是泛泛求关注？

每发现一个问题，必须用删除、调序、换用另一段原句或拆成另一条视频来解决；禁止润色补洞。

## 5. 情绪与伦理边界

- 优先保留真实情绪峰值，但不要只剪哭泣而抹掉前因后果。
- 金句后的笑、停顿、叹气和他人反应可能是节奏的一部分，剪辑单中明确保留尾巴。
- “对立面”不得被 AI 发明；需要由原文立场支持。
- 钩子可以前置，不能拼出受访者未表达的因果或承诺。
- 纠正明显事实错误前先回听、查证，并由用户决定是否加屏幕更正；不可替受访者补录新立场。
